# VoiceTemp Plugin

Плагин для автоматического создания временных голосовых каналов в Discord.

## Функции

- Автоматическое создание временных голосовых каналов при входе пользователя в канал-триггер
- Автоматическое удаление пустых временных каналов
- Настройка через slash команды
- Просмотр информации о настройках

## Использование

### Настройка

1. Используйте команду `/voicetemp-setup` для настройки:
   - Укажите канал-триггер (голосовой канал, при входе в который создается временный канал)
   - (Опционально) Укажите категорию, в которой будут создаваться каналы

2. Когда пользователь заходит в канал-триггер, автоматически создается временный канал

3. Когда пользователь покидает временный канал, он автоматически удаляется (если включено)

### Команды

- `/voicetemp-setup` - настройка канала-триггера и категории
- `/voicetemp-create` - создание временного канала вручную
- `/voicetemp-delete` - удаление текущего временного канала
- `/voicetemp-info` - просмотр информации о настройках

## Разработка

Этот плагин демонстрирует:
- Реализацию trait `Plugin`
- Регистрацию обработчиков взаимодействий
- Регистрацию обработчиков голосовых событий
- Регистрацию команд с опциями
- Работу с конфигурацией
- Использование HTTP клиента Discord

## Сборка

```bash
cargo build --release
```

Скомпилированная библиотека будет в `target/release/voicetemp_plugin.dll` (Windows) или `voicetemp_plugin.so` (Linux).

## Конфигурация

Плагин сохраняет конфигурацию в JSON файл через `PluginContext`. Настройки включают:
- ID канала-триггера
- ID категории для создания каналов
- Шаблон имени канала
- Автоматическое удаление пустых каналов
